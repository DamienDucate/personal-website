# Ducate — Portfolio

Personal portfolio site for Damien — web design, app development, and AI side projects.

**Live:** [ducate.com](https://ducate.com)

## Contents

- Portfolio homepage with terminal-inspired design
- [RusDucate](/rus/) — Russian learning PWA with flashcards and grammar drills
- [Vite Spot](/vitespot/) — archive memorial of vitespot.com
- [AppDucate](/appducate/) — archive memorial of appducate.com

## Deploy

Static HTML hosted on GitHub Pages with a custom domain (`CNAME`).

```bash
python3 -m http.server 8091
```

Sub-sites live at `/rus/`, `/vitespot/`, and `/appducate/` under the same root.
